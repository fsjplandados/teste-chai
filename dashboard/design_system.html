Diretrizes de Design

  • Espaçamento: Use múltiplos de 4px ou 8px para manter a consistência.
  • Bordas: Arredondamento padrão para cards é 14px. Para botões e inputs, 8px.
  • Sombra: Use sombras suaves (0 2px 14px rgba(0,0,0,0.06)) para criar profundidade sem poluir visualmente.
  • Hierarquia: Use variações de peso (Inter 800 vs 400) para guiar o olhar do usuário.
